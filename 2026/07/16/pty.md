A pty is a pseudoterminal, a software interface in Unix-like operating systems that emulates a
physical hardware terminal. They're to gain access to the shell.

A pseudo-TTY is a pair of character special files, a master file and a corresponding slave file
for bi-directional, interprocess communication[^1].

[^1]: Z/OS. 22 Mar. 2021, https://www.ibm.com/docs/en/zos/2.2.0?topic=files-pseudoterminal.
